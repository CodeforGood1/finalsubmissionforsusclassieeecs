# Sample Files for Sustainable Classroom LMS

This folder contains sample educational content for testing the bulk module upload feature.

## Folder Structure

```
samples/
├── science/
│   ├── pdfs/          # Sample science PDF documents
│   └── videos/        # Sample science video files
├── computer-science/
│   ├── pdfs/          # Sample computer science PDF documents
│   └── videos/        # Sample computer science video files
└── README.md          # This file
```

## How to Use

1. **Add your PDF files** to the respective `pdfs/` folders
2. **Add your video files** to the respective `videos/` folders
3. **Use the Bulk Module Steps Upload** feature in the teacher dashboard
4. Select files from these folders to create modules

## Recommended File Formats

### PDFs
- File format: `.pdf`
- Max size: 50MB per file
- Content: Lecture notes, study materials, worksheets

### Videos
- File formats: `.mp4`, `.webm`, `.ogg`, `.mov`
- Max size: 500MB per file
- Content: Lectures, demonstrations, tutorials
